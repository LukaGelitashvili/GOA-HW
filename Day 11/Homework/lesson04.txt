პირობითი განცხადებები (Conditional Statements)

პირობითი განცხადებები პროგრამირებაში ლოგიკური სტრუქტურებია, რომლებიც საშუალებას გვაძლევს, კოდის გარკვეული მონაკვეთები მხოლოდ მაშინ შესრულდეს, თუ დაკმაყოფილებულია კონკრეტული პირობა. ეს საშუალებას გვაძლევს, მივიღოთ გადაწყვეტილებები პროგრამის შიგნით არსებული პირობების მიხედვით.

პირობითი განცხადებების გამოყენებით პროგრამას შეუძლია განსაზღვროს, რომელ კოდის ბლოკს შეასრულებს კონკრეტული სიტუაციის დროს. პირობითი განცხადებები მოიცავს შემდეგ ელემენტებს:

1. if – გამოიყენება, რათა შეამოწმოს, შესრულებულია თუ არა გარკვეული პირობა. თუ პირობა სიმართლეა, პროგრამა ამ კოდის ბლოკს შეასრულებს.
     python
   age = 18
   if age >= 18:
       print("თქვენ ხართ სრულწლოვანი")
elif – გამოიყენება, როდესაც რამდენიმე პირობის შემოწმება გვჭირდება. თუ პირველი პირობა არ შესრულდა, პროგრამა შეამოწმებს elif-ით მოცემულ პირობას.

score = 75
if score > 90:
    print("შეფასება A")
elif score > 80:
    print("შეფასება B")
else:
    print("შეფასება C")
    else – გამოიყენება, თუ ყველა წინამდებარე პირობა ცრუ აღმოჩნდა, ანუ მხოლოდ მაშინ შესრულდება, როცა სხვა პირობები არ შესრულდა.
    
    temperature = 25
if temperature > 30:
    print("ცხელა")
else:
    print("ნორმალური ტემპერატურაა")

პირობითი განცხადებების გამოყენება პროგრამაში მნიშვნელოვანია, რადგან ის პროგრამის მოქნილობას და ადაპტირებას უზრუნველყოფს. მათ გარეშე, პროგრამას არ შეეძლება დინამიური გადაწყვეტილებების მიღება.
